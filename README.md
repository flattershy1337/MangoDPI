# MangoDPI
**MangoDPI - твой личный маршрутизатор в консоли MangoDPI**

**MangoDPI** - это форк утилиты zapret-discord-youtube, превращённый в мощный конструктор обхода блокировок. Всё управление — из единого BAT-файла, без лишних окон.

**Что умеет MangoDPI?**

 🥭 Единый консольный интерфейс:
  Один BAT-файл = полный контроль. Запустил - и сразу видишь меню со всеми возможностями.

 ****🧠 Создание собственных стратегий обхода****
 
  Собирай стратегии как конструктор:
   Выбирай тип фальсификации,
   Задавай TTL, размер пакетов,
   Указывай, что делать с UDP, TLS,
   Добавляйте название и описание.


**🕶️ Скрытие IP через консоль**

Встроенные методы маскировки реального IP.
  IP-ID Zero (Google-фильтр),
  TTL обман

**📋 Управление доменными списками**
Белые и чёрные списки доменов — прямо в BAT. Можно быстро добавлять/удалять домены, применять исключения для отдельных сервисов.
